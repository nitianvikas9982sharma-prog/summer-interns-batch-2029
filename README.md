# summer-interns-batch-2029
1. Clone the repository using Git CLI.
2. Create a new branch with your GitHub username.
3. Create a folder with the same name as your GitHub username.
4. Add a Markdown (`.md`) file inside the folder.
5. Write your introduction, achievements, skills, and anything else you want in the Markdown file.
6. Explore Markdown formatting like headings, lists, tables, images, GIFs, code blocks, and links.
7. Commit your changes using Git CLI.
8. Push your branch to GitHub.
9. Create a Pull Request (PR) to merge your branch into the main repository.
